# Tairon – Plataforma de cotización de PCs

Tairon es una plataforma web diseñada para ayudar a los usuarios a cotizar y armar su computador ideal según su presupuesto, priorizando rendimiento, compatibilidad de componentes y claridad en la elección.

La aplicación busca simplificar el proceso de selección de hardware, ofreciendo una experiencia intuitiva y accesible para usuarios sin conocimientos técnicos avanzados.

---

## Características

* Cotización de PCs según presupuesto
* Interfaz clara y enfocada en el usuario
* Diseño moderno orientado a conversión
* Base preparada para integración con lógica de recomendación

---

## Tecnologías utilizadas

* React
* Vite
* JavaScript
* Tailwind CSS
* PostCSS

El frontend está desarrollado con React utilizando Vite como herramienta de construcción, lo que permite un entorno de desarrollo rápido y optimizado.

El diseño está implementado con Tailwind CSS, facilitando la creación de interfaces modernas, limpias y responsivas.

La arquitectura del proyecto está basada en componentes reutilizables, permitiendo una estructura clara, mantenible y preparada para escalar.

El sistema está diseñado para integrarse con backend y evolucionar hacia funcionalidades más avanzadas como lógica de cotización y automatización.

---

## Objetivo

Construir una plataforma que permita a cualquier usuario armar su computador ideal sin necesidad de conocimientos técnicos avanzados, asegurando un equilibrio entre presupuesto y rendimiento.

Además, el sistema está pensado para evolucionar y crecer junto a las necesidades del usuario y del negocio.
